# Zonnect - Real-Time Chat App (BETA)

Zonnect is a modern real-time chat application built using the MERN stack. It focuses on simplicity, performance, and a clean user experience.

This project is currently in BETA, so features are still being developed and improved.

🌐 Live Demo: https://zonnect.vercel.app

---

## 🟢 Features

- User authentication using JWT (JSON Web Tokens)
- Login and signup system
- One-to-one messaging
- Clean and responsive UI
- Cloudinary integration (base setup for media)

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- DaisyUI
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB (MERN stack)
- JWT Authentication
- Arcjet

### Deployment
- Vercel (frontend)
- Render / Node hosting (backend)

## Status

This project is in BETA:
- Features may be incomplete
- UI/UX is still being improved
- Stability and performance updates are ongoing
